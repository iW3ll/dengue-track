# dengue-track
Track dengue statistics

## Requirements

•Python 3  
•pandas
- install packages in requirements.txt using pip.  
``pip install -r requirements.txt``

## How to use

````
python3 track.py

````
Or you can click on ``windows.bat`` or ``linux.sh`` according to your system.
